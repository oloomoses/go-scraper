# go-scraper
Web scraping fundamentals with golang

Scrapper build with golang and nethttp. Scraps 1000 books in under 4 seconds from https://books.toscrape.com/

## Built With
- Golang


### Prerequisites
Golang

### Setup
Run the the following commands in your terminal to setup the project in your local computer

1. Clone the project
```console
git clone git@github.com:oloomoses/go-scraper.git
```
2. Go to the project folder
```console
cd go-scraper
```
3. Check out the working branch
```console
git checkout features
```
4. run bundle to install packages
```console
go mod tidy
```

### Usage
Run the following command to run the project
```console
go run main.go
```
